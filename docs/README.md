# 文档索引

本目录包含产品开发过程中使用的标准模板、规范和流程文档。

## 模板

| 文件 | 用途 | 使用场景 |
|------|------|----------|
| [prd-template.md](templates/prd-template.md) | PRD 标准模板 | 新功能/优化项的需求定义 |
| [user-insight-report-template.md](templates/user-insight-report-template.md) | 用户洞察报告模板 | 用户研究结论的正式输出 |
| [design-review-checklist.md](templates/design-review-checklist.md) | 硬件设计评审 Checklist | 各阶段设计评审的逐项核查 |
| [project-execution-sop.md](templates/project-execution-sop.md) | 项目执行标准流程 | 项目从概念到量产的阶段管理 |
| [product-requirements-management-standards.md](templates/product-requirements-management-standards.md) | 需求管理规范 | 需求 ID、优先级、追溯、变更控制 |

## Jira 集成

| 文件 | 用途 |
|------|------|
| [wbs-to-jira-mapping.md](jira/wbs-to-jira-mapping.md) | PRD WBS → Jira Epic/Story/子任务 映射规则与对照表 |
| [jira-api-setup-guide.md](jira/jira-api-setup-guide.md) | Jira Cloud REST API 配置与使用指南 |

## 快速导航

- **要写 PRD** → 复制 `templates/prd-template.md`
- **要做用户研究** → 复制 `templates/user-insight-report-template.md`
- **要评审设计** → 打印 `templates/design-review-checklist.md`
- **要启动项目** → 参考 `templates/project-execution-sop.md` 的阶段门径
- **要管理需求** → 遵循 `templates/product-requirements-management-standards.md`
- **要同步 Jira** → 参考 `jira/wbs-to-jira-mapping.md`
