## 1. 执行摘要

**问题陈述：** Mars 5 Ultra 用户在长时间打印场景下反复报告机身底部发烫（实测42.5°C），明确表达了对设备寿命的担忧。这与该产品"省心全自动"的核心价值主张直接冲突——产品热设计未能覆盖最典型的使用工况。

**解决方案：** 从底部风道重新设计、风扇规格升级、固件PWM温控策略、关键元件隔热四个维度入手，在不大幅增加BOM成本（≤¥15）的前提下，将外壳最高温度从42.5°C降至≤38°C。

**成功指标：**
- 环境28°C条件下连续打印4小时，外壳底部温度 ≤ 38°C
- 满速噪音 ≤ 35dB(A)
- 用户盲测 ≥ 80% 报告"温而不烫"
- BOM增量 ≤ ¥15

## 2. 产品概述

- **产品型号：** Mars 5 Ultra（在产型号改进，非新型号）
- **产品线归属：** 消费级LCD光固化3D打印机
- **核心价值主张：** 全自动免调平、高精度打印、入门级智能旗舰
- **当前问题定位：** 热设计裕度不足，连续长时打印工况下外壳温度超出用户舒适阈值
- **改进范围：** 散热子系统局部优化，外壳/PCB/电源架构不变

## 3. 目标用户与使用场景

**用户画像：**

| 角色 | 描述 | 核心痛点 |
|------|------|----------|
| 进阶玩家 | 25-35岁，已有3D打印使用经验，经常连续打印多个模型 | "不敢无人值守"、"担心内部元件老化" |
| 小型工作室 | 使用 Mars 5 Ultra 做小批量生产，日均运行6-8小时 | 设备可靠性直接影响交付能力 |

**使用场景：**

| 场景 | 描述 | 散热要求 |
|------|------|----------|
| 常规打印 | 单次打印1-2小时，室温≤28°C | 外壳热点≤38°C |
| 连续长时打印 | 连续工作4小时以上，打印多个模型 | 温度不持续上升，达到稳态后保持在安全区间 |
| 极端环境 | 无空调夏日，室温32°C | 设备正常运行，无重启/关机/过热保护触发 |

## 4. 功能需求（MoSCoW优先级）

### Must-Have

| 功能模块 | 功能描述 | 验收标准 |
|----------|----------|----------|
| 底部风道优化 | 重新设计进风口、内部导流结构，确保气流覆盖主板核心发热区与LED驱动板背面 | CFD仿真显示PCB板背温降幅≥5°C；手板样机实测验证 |
| 风扇规格升级 | 评估同尺寸更高风压风扇型号（空间不可扩容时优先高风压） | 新风扇风量提升≥20%，满速噪音≤35dB | 
| 固件PWM温控 | 基于机身多点热敏电阻反馈动态调节风扇转速；打印结束延时散热 | 打印结束后3分钟内温度开始下降；无任务时风扇不转或最低转速 |

### Should-Have

| 功能模块 | 功能描述 | 验收标准 |
|----------|----------|----------|
| 元件隔热 | 对LED矩阵与外壳接触面增加隔热垫，减少热传导至触摸面 | 相同工况外壳热点额外降低2°C |

### Could-Have

| 功能模块 | 功能描述 |
|----------|----------|
| 温度状态指示 | 机身LED或APP界面显示当前温度状态（正常/偏高/需冷却） |

### Won't-Have（本版本明确不做）

- 外壳整体重新开模（仅允许底部局部修改）
- 液冷或半导体制冷等非常规散热方案
- PCB布局变更
- 电源模块外置方案

## 6. 设计约束

| 约束类别 | 具体要求 |
|----------|----------|
| 尺寸 | 风扇尺寸兼容现有安装位（若空间不可扩容，选同尺寸高风压型号） |
| 电气 | 不可变更PCB布局；风扇供电使用现有接口 |
| 结构 | 仅允许底部局部修改模具；外壳整体不开新模 |
| 环境 | 设计上限按32°C环境温度（模拟无空调夏日） |
| 采购 | 优先现有供应商产品线中的升级型号 |
| 兼容性 | 散热改动不可影响UV遮光性能（光固化核心功能） |

## 8. 验证与测试计划

采用 **Given-When-Then** 验收标准格式：

```
功能：连续打印长时温度控制
AC1：Given 环境温度稳定在28°C，When 连续打印4cm模型×3共4小时，
     Then 底部外壳温度 ≤ 38°C

功能：打印结束自动散热
AC2：Given 打印任务已完成，When 打印结束信号触发，
     Then 风扇继续运转，3分钟内机身温度开始下降

功能：噪音约束
AC3：Given 风扇满速运转，When 在距离设备1米处测量，
     Then A加权噪音 ≤ 35dB

功能：用户主观验证
AC4：Given 5位内部用户，When 连续打印4小时后触摸机身，
     Then ≥80% 报告"温而不烫"
```

**测试用例矩阵：**

| 测试用例ID | 测试场景 | 测试条件 | 预期结果 | 优先级 |
|------------|----------|----------|----------|--------|

| TC-HEAT-002 | 打印结束自动散热 | 完成打印任务后 | 3分钟内温度下降 | P0 |
| TC-HEAT-003 | 噪音测试 | 距离1米，满速状态 | ≤35dB(A) | P1 |

## 10. 固件接口定义

```json
{
  "command": "fan_speed",
  "params": {
    "target_rpm": 3200,
    "ramp_duration_ms": 2000,
    "trigger_source": "thermal_loop",
    "current_temp": {
      "board": 52.1,
      "led_back": 61.3,
      "ambient": 27.5
    }
  }
}
```

`trigger_source` 枚举值：
- `thermal_loop` — 正常温控循环
- `manual` — 用户手动设置
- `post_print` — 打印结束后延时散热
- `error` — 温度异常保护
