# Mars 5 Ultra 散热系统优化专项 PRD

| 文档版本 | 修改日期 | 修改人 | 修改说明 |
|----------|----------|--------|----------|
| V1.0 | 2026-05-07 | [产品经理] | 初稿，基于用户反馈与工程推演 |

## 1. 背景与问题定义

### 1.1 背景
Mars 5 Ultra 自2024年发布后，以全自动免调平、高精度打印获得市场认可。但在用户长期使用反馈中，"散热不足导致机身发烫"成为出现频率最高的可靠性相关抱怨。两位独立用户在不同环境下均报告了"长时间工作底部发烫明显"，并明确表达了"担心机器寿命"的焦虑。这与Mars 5 Ultra "省心全自动"的核心价值主张直接冲突。

### 1.2 问题定义
- **现状**：室温26℃环境中连续打印2小时，机身底部实测温度达42.5℃（用户测温枪数据），侧面39.2℃。
- **用户感知**："不敢无人值守"、"担心内部元件老化"。
- **本质**：热设计未能覆盖用户最典型的"连续长时打印"工况，导致产品可靠性的用户可感知度不达标。

## 2. 产品目标

- **核心目标**：在环境温度28℃条件下，连续打印4小时后，机身外壳最高温度点（底部）≤ 38℃，消除用户"烫手"体感。
- **辅助目标**：
  - 内部关键元件（LED矩阵、主板电源IC）温度控制在规格书推荐范围内
  - 风扇转速策略在温控与噪音间取得平衡（满速噪音 ≤ 35dB）
  - 不显著增加BOM成本（单台增量 ≤ ¥15）

## 3. 范围与约束

### In Scope
- 底部散热风道重新设计（含进风口、内部导流、出风口）
- 风扇规格升级（尺寸兼容性评估、高风压型号选型）
- 固件 PWM 温控策略（含打印结束延时散热）
- 关键元件隔热（LED 矩阵与外壳接触面）
- DVT 热测试与噪音测试

### Out of Scope
- 外壳整体重新开模（仅允许底部局部修改）
- 液冷或半导体制冷等非常规散热方案
- PCB 布局变更
- 电源模块外置方案

### 假设与约束
- 假设现有风扇供应商产品线中存在同尺寸更高风压型号
- 受限于当前 PCB 布局不可变更，导流结构调整需避开现有元件位置
- 环境温度上限按 32℃ 设计（模拟无空调夏日）
- 噪音目标 ≤ 35dB 为 A 加权 1 米处测量值

## 4. 功能需求

### 4.1 底部散热风道优化
- **说明**：重新设计底部进风口、内部导流结构，确保气流覆盖主板核心发热区与LED驱动板背面。
- **验收标准**：CFD仿真显示底部PCB板背温降幅≥5℃；手板样机实测验证。

### 4.2 风扇规格升级
- **说明**：评估更大尺寸或更高转速风扇的兼容性（若空间不可扩容，则选型同尺寸更高风压型号）。
- **验收标准**：新风扇风量提升≥20%，满速噪音≤35dB。

### 4.3 固件智能温控策略
- **说明**：基于机身多点热敏电阻反馈，动态调节风扇转速。打印结束后继续运行风扇直至温度降至阈值。
- **验收标准**：打印任务结束后3分钟内，外壳温度开始下降；无任务时风扇不转或最低转速。

### 4.4 关键元件隔热
- **说明**：对LED矩阵与外壳接触面增加隔热垫，减少热传导至触摸面。
- **验收标准**：相同工况下外壳热点温度额外降低2℃。

## 5. 非功能需求

- **噪音约束**：距离设备1米处测量，满速运行噪音 ≤ 35dB(A)，低速 ≤ 28dB(A)
- **可靠性**：新风扇MTBF ≥ 20000小时，新增隔热材料不变形、不释放异味
- **兼容性**：散热改动不可影响当前料槽安装、平台行程、UV遮光性能
- **用户主观验证**：邀请 5 位内部用户连续打印 4 小时后盲测触摸感受，≥80% 报告"温而不烫"

## 6. 数据结构与固件接口定义

```json
{
  "command": "fan_speed",
  "params": {
    "target_rpm": 3200,
    "ramp_duration_ms": 2000,
    "trigger_source": "thermal_loop",
    "current_temp": {
      "board": 52.1,
      "led_back": 61.3,
      "ambient": 27.5
    }
  }
}
```

`trigger_source` 枚举值：
- `thermal_loop` — 正常温控循环
- `manual` — 用户手动设置
- `post_print` — 打印结束后延时散热
- `error` — 温度异常保护
