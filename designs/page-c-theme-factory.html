问题诊断

三大核心痛点

从用户反馈中提炼的结构化问题定义

P0 · 散热信任危机

底部温度过高引发安全与可靠性担忧

实测底部温度达到 42.5°C，超出用户舒适阈值。长期高温可能导致 LCD 屏幕老化加速、UV 光源衰减，进而影响打印精度与设备寿命。用户对长时间打印的安全性产生疑虑。

P1 · 结构交互连锁

料槽拆装牵涉排线，操作繁琐且易损坏

取盖子→拔线→卸料槽的冗余步骤，在清理和维护场景中反复出现。LCD 排线接口非热插拔设计，频繁操作增加引脚弯曲和接触不良的风险。

范围定义

项目范围

明确边界，聚焦可交付价值

✓ In Scope

  • ✓ 底部主动散热风道设计 & CFD 仿真验证
  • ✓ 料槽磁吸快拆结构（替代螺丝+排线方案）
  • ✓ 机壳模具修改：增加加强筋 + 哑光纹理表面
  • ✓ 低气味树脂槽材料选型与供应商切换
  • ✓ 噪音优化：风扇选型 + 减震垫 + PWM 调速
  • ✓ 盲测验证：20 人样本，双盲温度体感测试

✗ Out of Scope

  • — UV 光源模组升级（非散热根本原因，且涉及安全认证）
  • — 外观 ID 大改（模具投入过高，纳入下一代规划）
  • — Z 轴机械精度提升（与散热无关，独立立项）
  • — Wi-Fi 模块追加（硬件架构不支持，需 SoC 升级）
  • — 加热温控系统（需求优先级低，二期评估）
